# Demos

Demos是用于练手Java各种库的项目，使用maven构建，目前包含
* activeMQ
* aspectj
* 百度OCR
* jetty
* jgroups 群组通讯
* HttpClient
* Java：部分API，如classLoader，压缩，swing，线程池等
* JSON解析：Fastjson, Gson, SfJSON等
* Log4j 
* msLinks java快捷方式
* nashorn java js引擎
* FlexMark：MarkDown文本转HTML
* paho：MQTT消息的java实现
* 设计模式
* Spring 
* vfs：apache文件系统
* xstream
